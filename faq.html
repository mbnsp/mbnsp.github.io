Mam problem z instalacją TBuntu 22 (wątek dotyczy również wersji DEMO i emulatorów)

Przede wszystkim sprawdź plik readme.md lub readme.write. Później upewnij się, że masz środowisko JRE (Java Runtime Environment) w odpowiedniej wersji. W tym celu otwórz CMD.exe lub Terminal i wpisz: java -version. Jeżeli wyskoczy Ci błąd to znaczy, że musisz pobrać JRE. Można to zrobić ze strony www.java.com. ​ W przypadku innego problemu prosimy wypełnić formularz kontaktu i w sekcji: czego dotyczy wiadomość, napisać: TBuntu Bug Report. Aby rozpatrzyć problem potrzebujemy jak najwięcej informacji takich jak:
  • Specyfikacja komputera
  • Zainstalowany system Macierzysty (np. Windows, MacOs, Linux, MIS-DOS)
  • Wersja Java Runtime
  • Połączenie z internetem.

​​ Nie podając konkretnych informacji możemy nie rozwiązać problemu. Na odpowiedź przewidujemy maks. 14 dni roboczych (nie dotyczy wyjazdów oraz innych nie obecności operatorów lub członków M.B.-Interactive Software)
